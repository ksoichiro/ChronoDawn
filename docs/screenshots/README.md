# Chrono Dawn Screenshots

This directory contains high-quality screenshots for CurseForge and Modrinth galleries.

## Naming Convention

Screenshots follow the pattern: `NN-description.png`

- `NN`: Two-digit sequence number (01-14)
- `description`: Brief description in kebab-case (lowercase, hyphens)
- Extension: `.png` (PNG format required)

### Examples
- `01-ancient-ruins.png`
- `02-portal-activation.png`
- `03-chrono-dawn-dimension.png`

## Technical Requirements

- **Resolution**: 1920x1080 (16:9 aspect ratio)
- **Format**: PNG
- **File Size**: <5MiB (Modrinth compatibility)
- **HUD**: Hidden (press F1 before capturing)

## Recommended Mods for Quality

- **Iris Shaders** - Visual enhancement (recommended: Complementary Shaders)
- **Sodium** - Performance optimization while capturing
- **Replay Mod** - Cinematic camera angles and smooth camera paths
- **Distant Horizons** - Extended render distance for landscape shots

## Capture Guidelines

1. **Disable HUD**: Press `F1` before capturing
2. **Optimal Lighting**: Capture during in-game day or use shaders for ambient lighting
3. **Frame Composition**: Follow rule of thirds, leave space for text overlays
4. **Action Shots**: Capture bosses mid-attack for dynamic screenshots
5. **Variety**: Mix landscapes, structures, action shots, and items

## Quality Checklist

Before uploading, verify each screenshot:

- [ ] Resolution is exactly 1920x1080
- [ ] HUD is hidden (no hotbar, health bar, etc.)
- [ ] File size is under 5MiB
- [ ] Image is sharp and well-lit
- [ ] No debug information visible (F3 screen)
- [ ] Represents mod features accurately
